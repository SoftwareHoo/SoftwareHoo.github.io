Projects

Word of Mouth

Word of Mouth creates a combination of social media and a recipe website, but removes all of the unnecessary background infromation and gets straight to the ingredients and steps. This is a group project built in UVA's Advanced Software Development course. This project was built primarily with Django, JavaScript, JQuery, and PostgreSQL. It uses Google's API to allow for Google sign-in, and Amazon AWS S3 API for image storage, among other things.

PetPals

PetPals is storage system and organizer for the biggest pet owners. This project was created in collaboration with Dan Do at UVA. It was built primarily with PHP, JavaScript, JQuery, and MariaDB.

Space Shooter

Space Shooter is a game created in collaboration with Cooper Yurish at UVA. This project was built in an introductory programming course using Python 2.7 and the PyGame library.
